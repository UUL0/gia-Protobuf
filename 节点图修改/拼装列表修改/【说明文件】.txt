批处理拖放顺序
数字按照NNN~000的顺序鼠标选择才会按顺序000~NNN执行

从NNN~000选择，并从第000文件长按拖放到cmd文件上才是顺序的000~NNN




----------浮点数拼装列表，批量生成
[批量]_[拖放]生成拼装浮点.cmd


多个文件、任意行数的浮点数，按100行进行切分生成N个列表

需修改[批量]_[拖放]生成拼装浮点.cmd内的参数定义
TID是节点图内部的对象顺序ID，CID是可能是内部ID，是唯一的，用一个空节点放一个拼装列表，这样方便查看ID与导入
set TID=2
set CID1=1
设置基础坐标，列表在节点图内的起始位置
set CX=0
set CY=0
坐标增量，每个列表的间隔，这个是相对于单文件的内容的增量
set Xstp=500
set Ystp=0
